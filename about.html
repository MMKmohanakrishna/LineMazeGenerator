About Line Maze Generator

Line Maze Generator is a client-side web application for generating and exporting mazes in several visual styles (orthogonal, hexagonal, triangular). It focuses on producing clean maze connectivity data and images for use in games, puzzles, or learning projects.

Author: Malenati Mohana Krishna

Contact: echostepgame@gmail.com

Features

  • Orthogonal (square), Sigma (hexagonal) and Delta (triangular) visual styles
  • Multiple generation algorithms (Recursive Backtracker, Prim's, Kruskal's)
  • Biased generation controls (Elitism E, River R, Confusion C)
  • Export as PNG image and maze connectivity text

Usage

Open index.html (the main page) and choose your desired size, style and algorithm. Click "Generate New Maze" to render, then export using the download buttons.
